DNS Spoof Attack

Para llevar a cabo un ataque de DNS spoofing, siga los pasos a continuación utilizando Kali Linux:

  1. Abrir Kali Linux: Inicia sesión en Kali Linux para comenzar.
  2. Ejecutar el comando en la terminal: Abre la terminal y ejecuta el siguiente comando:
    3. sudo /etc/ettercap/etter.conf Este comando abrirá el archivo de configuración de Ettercap, donde realizaremos algunas modificaciones.
  3. Modificar los parámetros en el archivo de configuración: En el archivo de configuración de Ettercap, busca los parámetros "ec_uid" y "ec_gid" y configúralos a 0.
  4. Habilitar los comandos de redirección en el archivo de configuración: Busca la sección "redir_command_on" en el archivo y descomenta los comandos que se encuentran en la sección de Linux.
  5. Guardar y salir del archivo de configuración: Guarda los cambios realizados en el archivo y ciérralo.
  6. Configurar el archivo hosts: Ahora debemos configurar el archivo de hosts, que determinará a qué IP haremos el DNS spoofing. Puedes editar el archivo /etc/ettercap/etter.dns para agregar los nombres de dominio y las IP a las que deseas redirigir las solicitudes de DNS.
  7. Guardar y salir del archivo de hosts: Guarda los cambios realizados en el archivo de hosts y ciérralo.
  8. Abrir Ettercap: Abre el programa Ettercap en Kali Linux.
  9. Iniciar el sniffing en la interfaz deseada: Configura la interfaz de red en la que deseas realizar el sniffing para capturar el tráfico de red.
  10. Detener el programa: Detén el programa para realizar las configuraciones necesarias antes de continuar.
  11. Abrir la sección de plugins y activar el dns_spoofing: Dentro de Ettercap, accede a la sección de plugins y activa el plugin "dns_spoofing".
  12. Escanear los hosts: Realiza un escaneo para identificar los hosts presentes en la red.
  13. Añadir los hosts objetivo: Selecciona los hosts a los que deseas realizar el DNS spoofing y agrégalos como targets en Ettercap.
  14. Iniciar el ARP poisoning: Inicia el ataque de ARP poisoning para interceptar el tráfico de red entre los hosts y el servidor DNS.
  15. Iniciar el programa: Inicia el programa Ettercap para comenzar el DNS spoofing.
  16. Observar los resultados: Puedes verificar los resultados realizando un nslookup en la consola para comprobar que las solicitudes de DNS están siendo redirigidas correctamente.
